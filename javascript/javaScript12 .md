## Javascript Events

Aapne onClick event ke baarein mei padha hoga. Aise hi bahut saare events hote hai jaise **onchange, onclick, onmouseover, onmouseout, onkeydown, onload.** 
onclick ko badle
- **onmouseover** se
- **onmouseout** se
- **onkeydown** se
Har event kisi html element se **bind** (yaani attach ya judta) hota hai. Jaise yeh events button se **bind** hue hai. Toh **onkeydown** ko check karne se pehle, button par click karo and phir koi bhi key press karke dekho, usse yeh event **trigger** (yaani start/shuru) hoga.
